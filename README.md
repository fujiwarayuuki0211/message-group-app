# アプリ名
  Message-Group-App

# 概要
  このアプリケーションは、家族、仕事、友人、その他の使用用途に分けて、メッセージをやりとりすることが出来るアプリケーションです。
  このアプリケーションを利用するには、新規登録を完了することで新規メッセージグループを作成することが出来ます。作成されたグループに所属しているユーザーであれば、新規メッセージをやりとりする画面に遷移する事が出来ます。
  グループに所属するメンバーであれば、そのグループを削除する事が出来ます。
  トップページでは、使用用途に分けて、グループ情報を表示しています。

# 制作背景（意図）
  私はこのアプリケーションを、複数のメッセージアプリを併用するユーザーの手間を失くすために制作いたしました。
  多くのユーザーは、仕事用のメッセージアプリ、家族、友人用のメッセージアプリ、その他の目的のメッセージアプリと複数を目的別に利用している事があると思っております。その際、複数のメッセージアプリをユーザーは行き来する手間があります。私はその手間を一つのメッセージアプリで、メッセージ用途に分けるアプリケーション開発を行うことで、失くそうと考え、このアプリケーションを開発いたしました。

  ・メッセージをやり取りする相手が可視化できるように、新規メッセージ投稿画面にて、そのグループに所属するメンバーを表示した点。

# 課題や今後実装したい機能
## 課題
・ユーザー同士の関係を考慮されず、このアプリケーションに登録しているユーザー全員がグループメンバーになる対象になっている点。
・メッセージグループのカテゴリーが４種類だけと限定されている点。
・利用するユーザー情報が少なく、ユーザーの信用度が低い点。
・投稿したメッセージの投稿者の区別がわかりにくい点。
## 今後実装したい機能
・ユーザー同士を関係あるもののみに限定するために、グループメンバーを招待制にする機能。
・ユーザー自身で、新規作成するグループのカテゴリーを新たに追加できる機能。
・新規登録の際に、登録する情報を増やし、ユーザー情報の充実をさせる機能。
・新規メッセージ投稿画面にて、画面を閲覧しているユーザーが投稿したメッセージは、画面右側に表示する機能。


# テーブル設計

## users テーブル

| Column   | Type   | Options     |
| -------- | ------ | ----------- |
| nickname | string | null: false |
| email    | string | null: false |
| password | string | null: false |

### Association

- has_many :group_users
- has_many :groups, through: group_users
- has_many :messages

## groups テーブル

| Column      | Type    | Options     |
| ----------- | ------- | ----------- |
| name        | string  | null: false |
| category_id | integer | null: false |

### Association

- has_many :group_users
- has_many :users, through: group_users
- has_many :messages

## group_users テーブル

| Column | Type       | Options                        |
| ------ | ---------- | ------------------------------ |
| user   | references | null: false, foreign_key: true |
| group  | references | null: false, foreign_key: true |

### Association

- belongs_to :group
- belongs_to :user

## messages テーブル

| Column  | Type       | Options                        |
| ------- | ---------- | ------------------------------ |
| content | string     | null: false                    |
| user    | references | null: false, foreign_key: true |
| group   | references | null: false, foreign_key: true |

### Association

- belongs_to :group
- belongs_to :user
